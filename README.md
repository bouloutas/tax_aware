# Tax-Aware Portfolio Management

This repository contains tax-aware portfolio management tools and analytics:

- **barra/**: Barra risk model analytics
- **edgar/**: SEC filing analysis and data
- **fama_french/**: Fama-French factor analysis
- **parametric/**: Parametric portfolio construction
